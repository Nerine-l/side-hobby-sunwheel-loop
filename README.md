# side-hobby-sunwheel-loop

サブの趣味として、机の端でずっと流しておけるように作った Pygame の小さなループアニメーションです。

## Features

- 8秒で自然に一周する、キー入力なしの2Dアニメーション
- 画像素材なし、Pygameの図形描画だけで構成
- 背景の空色が朝日から昼、夕日、薄暮へなめらかに変化
- 元の輪の回り方を保った粒子リング
- リサージュ曲線の軌跡と、太陽の反射を重ねた穏やかな画面

## 実行方法

```powershell
pip install -r requirements.txt
python main.py
```

## メモ

授業課題Aの条件に合わせて、画面サイズは800x450、ループ長は8秒にしています。
